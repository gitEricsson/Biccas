# Biccas Landing Page

A modern, responsive landing page for Biccas, featuring a hero section, feature grids, pricing with interactive hover effects, and a contact form.

![Biccas Design](./public/Design.png)

## Features
- **Modern UI**: Clean design with vibrant accents and glassmorphism.
- **Interactive Pricing**: Dynamic hover effects on pricing cards.
- **Responsive Layout**: Optimized for various screen sizes.

## How to use
Open `index.html` in any modern web browser.
